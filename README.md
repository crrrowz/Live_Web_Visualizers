## 📖 نظرة عامة

**Voice Study Visualizer** هو تطبيق ويب تفاعلي مبني بتقنية **Three.js** يحوّل صوتك في الوقت الحقيقي إلى تصوّرات ثلاثية الأبعاد مذهلة. صُمم ليكون رفيقك أثناء الدراسة — يستمع لصوتك عبر الميكروفون ويحوّله إلى مشاهد بصرية حيّة تتفاعل مع نبرتك وحدّة صوتك.

يتضمن التطبيق **نظام تتبع دراسي كامل** مع مستويات، نقاط خبرة (XP)، سلسلة إنجازات يومية، وعبارات تحفيزية باللغة العربية.

## ✨ الميزات

### 🎨 مشاهد ثلاثية الأبعاد (8 مشاهد)

| المشهد | الوصف |
|--------|-------|
| 🌌 **Matter × Antimatter** | تصادم جسيمات مادة ومضادة مع تشوّه عضوي |
| ☀️ **Solar System** | نظام شمسي مصغّر بكواكب تدور |
| 🧬 **DNA Helix** | لولب الحمض النووي بجزيئات متوهجة |
| 🌀 **Galaxy** | مجرة حلزونية بنجوم وسُدم |
| 🧠 **Neural Network** | شبكة عصبية بعُقد ووصلات متحركة |
| 🌊 **Deep Ocean** | أعماق المحيط بكائنات بحرية مضيئة |
| 🚀 **Warp Tunnel** | نفق فضائي بسرعة الضوء |
| 🌈 **Aurora Borealis** | شفق قطبي بأضواء متموّجة |

### 📊 نظام الدراسة
- ⏱️ **مؤقت الجلسة** — يتتبع مدة كل جلسة دراسة
- ⭐ **نظام المستويات** — تكتسب 10 XP لكل دقيقة دراسة
- 🔥 **سلسلة الإنجازات** — تتبع أيامك المتتالية في الدراسة
- 💬 **عبارات تحفيزية** — اقتباسات عربية ملهمة تتغير تلقائياً
- 💾 **حفظ تلقائي** — يحفظ تقدمك حتى عند إغلاق المتصفح

### 🎮 التحكمات

| المفتاح | الوظيفة |
|---------|---------|
| `Space` (نقرة) | عكس الألوان |
| `Space` (مطوّل) | عرض لوحة الإحصائيات |
| `S` | حفظ الجلسة الحالية |
| `R` | إعادة تعيين الإحصائيات |
| `T` | التبديل بين المشاهد |

### 🔧 مميزات تقنية
- تأثيرات **Bloom/Glow** بمحرك ما بعد المعالجة
- تحليل صوتي بـ **Web Audio API** مع ضاغط ديناميكي
- **Shaders مخصصة** بـ GLSL لتشوّه الأسطح
- معمارية **موديلارية** — كل مشهد ملف مستقل
- واجهة **RTL** تدعم العربية بالكامل

## 🛠️ التثبيت والتشغيل

### المتطلبات الأساسية

- **Node.js** 18.0 أو أحدث — [nodejs.org](https://nodejs.org/)
- **متصفح حديث** (Chrome / Edge / Firefox)
- **ميكروفون** للتفاعل الصوتي

### خطوات التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/crrrowz/Live_Web_Visualizers.git

# 2. الانتقال لمجلد المشروع
cd Live_Web_Visualizers

# 3. تثبيت المكتبات
npm install

# 4. تشغيل الخادم
npm start
```

افتح متصفحك على: **http://localhost:3000**

> ⚠️ عند فتح التطبيق لأول مرة، سيطلب المتصفح إذن الوصول للميكروفون. اضغط **"السماح"** لتفعيل التفاعل الصوتي.

## ⚙️ الإعدادات

جميع الإعدادات في ملف **`public/js/config.js`**:

| الإعداد | القيمة | الوصف |
|---------|--------|-------|
| `micGain` | `0.5` | تضخيم الميكروفون |
| `bassMultiplier` | `2.5` | مضاعف استشعار الباس |
| `smoothingSpeed` | `0.1` | سرعة المتابعة البصرية |
| `bloomStrength` | `1.5` | شدة التوهج |
| `particleCount` | `2000` | عدد الجسيمات |
| `xpPerMinute` | `10` | نقاط خبرة لكل دقيقة |

## 🔌 واجهة API

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| `GET` | `/api/stats` | جلب إحصائيات الدراسة |
| `POST` | `/api/sessions` | حفظ جلسة (يحتاج `durationMin`) |
| `POST` | `/api/stats/reset` | إعادة تعيين الإحصائيات |
